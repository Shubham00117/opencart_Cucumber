# BDD Automation with Cucumber

## 🚀 Overview
Behavior-Driven Development (BDD) framework for testing OpenCart functionalities.

## 🛠️ Tech Stack
- **Tools:** Java, Selenium, Cucumber, JUnit
- **Design Pattern:** Page Object Model (POM)
- **Reporting:** HTML Reports / Allure Reports

## ✨ Key Features
- Gherkin feature files
- Step definition separation
- Cucumber HTML reports
- Readable test scenarios

## ⚙️ Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Shubham00117/opencart_Cucumber.git
   ```
2. Navigate to the project directory.
3. Install dependencies (refer to project config files).

## 📊 Usage
Run tests using the standard command-line tools for the tech stack (e.g., `npm test`, `mvn test`, or `pytest`).
